# Mohammad H. Masah | Portfolio

Bienvenue sur mon portfolio personnel. Ce site présente mes projets réalisés à **Epitech** et mes compétences en développement logiciel.

## 🚀 Projets en vedette

- **Klivio** : Plateforme d'apprentissage interactive.
- **Epitalk** : Application web front-end pour les étudiants.

## 🛠️ Technologies utilisées

- **Moteur de site** : Jekyll
- **Langages** : HTML, CSS, JavaScript, Markdown
- **Hébergement** : GitHub Pages

## ⚙️ Installation locale

Pour exécuter ce projet localement :

1. Clonez le dépôt.
2. Installez les dépendances avec `bundle install`.
3. Lancez le serveur avec `bundle exec jekyll serve`.
