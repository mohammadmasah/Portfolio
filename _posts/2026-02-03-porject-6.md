&#8203;**NEXUS** is a full-stack social networking platform built to bridge the gap between complex backend logic and a fluid, high-performance user interface. This project highlights my expertise in **React.js**, **Secure Authentication**, and building interactive community features.

### Key Features

- **Secure Authentication System:** A robust Login and Sign-up flow ensuring user data protection. Features include secure session management and protected routes to prevent unauthorized access.
- **Interactive Feed System:** A real-time post listing with advanced filtering, relative time formatting, and seamless content loading.
- **Full Engagement Suite:** Comprehensive **Like** and **Comment** systems with instant UI feedback (Optimistic UI updates) and interactive toggle views.
- **Dynamic Content Management (CRUD):** Smooth "In-place Editing" system where users can modify their posts without leaving the page, alongside soft-delete capabilities.
- **Personalized Experience:** Each user has a unique profile, and the system strictly enforces ownership—only the creator of a post or comment has the authority to edit or remove it.
- **Luxury UI/UX:** A mobile-first design utilizing **Tailwind CSS** for a clean, modern aesthetic with smooth transitions and glassmorphism elements.
